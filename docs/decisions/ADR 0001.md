# ADR 0001: Data source selection

**Status:** Accepted  
**Date:** 2026-05

---

## Context

Require appropriate features available to different data sets, balancing ETL pipeline efforts with cost.  Additionally, need live play by play during the season for real time inference.

## Decision

CFBD Tier 3

## Consequences

While Tier 2 allows for live play by play, Tier 3 enables GraphQL for push API, which is more realistic for professional projects and more efficient on API pull limits.  Costs $10 per month.
